# jesus-perspective

> 耶稣的思维框架与表达方式 — AI Skill

基于四福音书 + 3000行学术调研（6维度 × 31个独立来源），提炼6个核心心智模型、8条决策启发式和完整的表达DNA。

激活后，AI 将以耶稣在福音书中呈现的视角回应——用比喻、反问、悖论来启发思考。

## 安装

```bash
npx openskills install yuanhonglin/jesus-perspective
```

或手动克隆：

```bash
git clone https://github.com/yuanhonglin/jesus-perspective.git ~/.claude/skills/jesus-perspective
```

## 触发方式

在对话中说以下任何一句即可激活：

- 「用耶稣的视角看看这个问题」
- 「耶稣会怎么看」
- 「Jesus perspective」
- 「切换到耶稣」
- 「帮我用耶稣的角度想想」

## Skill 结构

```
jesus-perspective/
├── SKILL.md              # 主 Skill 文件（思维模型 + 表达DNA + 决策框架）
├── references/           # 学术调研资料
│   └── research/         # 6维度调研文档
└── scripts/              # 数据处理脚本
```

## 核心内容

- **6个心智模型**：天国临在、关系本体论、悖论思维、权力倒转、恩典经济学、末世紧迫感
- **8条决策启发式**：心灵优先、反问揭示、弱者优先、预言勇气、关系修复、丰盛信念、先行一步、言行合一
- **表达DNA**：比喻密度、反问频率、悖论张力、具象化程度等量化指标
