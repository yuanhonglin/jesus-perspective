# 耶稣的表达风格：语言DNA与修辞特征分析

> 研究说明：福音书以希腊文写成，耶稣原本使用亚兰文（及部分希伯来文）教导。以下分析的是「福音书作者呈现的耶稣表达风格」，包含翻译和编辑的层次。这恰恰是「耶稣的表达」如何被受众接收的方式。

## 1. 比喻（Parables）作为核心表达工具

### 1.1 总体统计

- **[福音书文本呈现]** 比喻约占耶稣教导记录的三分之一（来源：Wikipedia, Parables of Jesus）
- **[学者分析]** 按和合本标准分类，正典福音书共有 38-60 个比喻（因定义宽严而异）：
  - 1913年《天主教百科全书》计数为 33 个
  - 部分学者将「比喻性谚语」纳入，计数可达 60 个
- **[福音书文本呈现]** 各卷分布：
  - 路加福音：24 个（18 个独有）
  - 马太福音：23 个（11 个独有）
  - 马可福音：8 个（2 个独有）
  - 约翰福音：无标准比喻（仅有寓言式意象，如好牧人、葡萄树）

### 1.2 比喻类型与主题分类

**A. 天国/神国类（听见、寻求、成长）：**
- 撒种（太13:1-23；可4:1-20；路8:4-15）
- 隐藏的财宝（太13:44）
- 重价珍珠（太13:45-46）
- 生长的种子（可4:26-29）
- 芥菜种（太13:31-32；可4:30-32；路13:18-19）
- 面酵（太13:33；路13:20-21）

**B. 失落与救赎类：**
- 失羊（太18:12-14；路15:1-7）
- 失钱（路15:8-10）
- 浪子（路15:11-32）
- *路加福音中这三者构成「失落后救赎」三部曲*

**C. 爱与饶恕类：**
- 好撒玛利亚人（路10:25-37）
- 两个欠债人（路7:36-50）
- 不饶恕人的仆人（太18:23-35）

**D. 祷告类：**
- 半夜借饼的朋友（路11:5-8）
- 不义的法官（路18:1-8）
- 法利赛人与税吏（路18:9-14）

**E. 终末/审判类：**
- 忠心仆人（太24:45-51；路12:42-48）
- 十童女（太25:1-13）
- 大筵席（太22:1-14；路14:16-24）
- 无知财主（路12:16-21）
- 凶恶园户（太21:33-46；可12:1-12；路20:9-19）
- 稗子（太13:24-30）
- 撒网（太13:47-50）
- 无花果树发芽的比方（太24:32-35；可13:28-31；路21:29-33）
- 不结果的无花果树（路13:6-9）
- 葡萄园工人（太20:1-16）

### 1.3 比喻结构特征

- **[福音书文本呈现]** 比喻的标准结构：
  1. 引入用语（「天国好像...」「一个人有...」「你们想...」）
  2. 叙事（日常生活场景：农夫撒种、妇人找钱、父亲等儿子）
  3. 隐含或明示的属灵含义
  4. 有时以「有耳可听的，就应当听」收尾

- **[学者分析]** Adolf Jülicher 将比喻分析为三部分：
  - `Bildhälfte`（画面部分）：故事的具体要素
  - `Sachhälfte`（现实部分）：比喻指向的属灵/道德真理
  - `tertium comparationis`（比较点）：连接以上两者的核心类比点

- **[福音书文本呈现]** 比喻根植于希伯来传统中的 `mashal`（מָשָׁל，意为「比较」或「简短格言」）

### 1.4 分层策略：公开 vs 私下解释

- **[福音书文本呈现]** 耶稣明确采用分层策略：
  - **公开场合**：只讲比喻，不解释（太13:34：「若不用比喻，就不对他们说什么」）
  - **私下**：对门徒详细解释（可4:34：「私下里没有一样不对门徒讲解」）
  - 目的陈述：可4:11-12 ——「神国的奥秘只叫你们知道，若是对外人讲，凡事就用比喻，叫他们看是看见，却不晓得；听是听见，却不明白」

- **[我推断]** 这是一种**修辞过滤机制**：
  - 对愿意深究的人（门徒），比喻成为通往更深真理的通道
  - 对不愿意或不能理解的人（外人），比喻停留在故事层面
  - 这使得教导既有群众感染力，又有门徒训练的深度，一石二鸟

- **[学者分析]** 这呼应了犹太智慧传统（特别是箴言和传道书）中「智慧对愚昧人隐藏」的主题，但耶稣将其系统化为教学方法

### 1.5 经典比喻的表达特征矩阵

| 比喻 | 主题 | 长度 | 开门见山？ | 结尾反转？ | 需要解释？ |
|------|------|------|-----------|-----------|-----------|
| 撒种 | 听道状态 | 中 | 是 | 否 | 是（对门徒） |
| 浪子 | 悔改与恩典 | 长 | 是 | 是 | 否 |
| 好撒玛利亚人 | 真正的邻舍 | 中 | 否（用问题引出） | 是 | 否 |

| 十童女 | 警醒预备 | 中 | 是 | 是（迟到的被关） | 否 |

## 2. 悖论与反直觉表达

### 2.1 核心悖论清单（[福音书文本呈现]）

| 悖论语句 | 出处 | 类型 |
|---------|------|------|
| 「在前的将要在后，在后的将要在前」 | 太19:30, 20:16；可10:31；路13:30 | 地位翻转 |
| 「凡要救自己生命的，必丧掉生命；凡为我丧掉生命的，必得着生命」 | 太16:25；可8:35；路9:24；约12:25 | 生死翻转 |
| 「你们中间谁愿为大，就必做你们的用人；谁愿为首，就必做你们的仆人」 | 太20:26-27；可10:43-44 | 大小翻转 |
| 「施比受更为有福」 | 徒20:35（引用耶稣语） | 施受翻转 |
| 「有钱财的人进神的国是何等地难哪！骆驼穿过针的眼，比财主进神的国还容易呢」 | 太19:23-24；可10:25；路18:24-25 | 贫富翻转 |
| 「虚心的人有福了…哀恸的人有福了…温柔的人有福了…」 | 太5:3-12；路6:20-23 | 价值翻转 |
| 「为义受逼迫的人有福了」 | 太5:10 | 苦乐翻转 |
| 「爱你们的仇敌，为那逼迫你们的祷告」 | 太5:44；路6:27-28 | 敌友翻转 |
| 「不要与恶人作对。有人打你的右脸，连左脸也转过来由他打」 | 太5:39 | 强弱翻转 |

### 2.2 悖论频率与模式

- **[福音书文本呈现]** 悖论是耶稣最持久、最一以贯之的修辞模式，贯穿四福音书
- **[我推断]** 出现次数：至少 25-30 处明显的悖论/翻转语句（保守估计），分布在所有福音书来源中
- **[学者分析]** 这种「翻转世界」的修辞根植于犹太先知传统（如以赛亚书「豺狼必与羊羔同卧」），但耶稣将其扩展到日常伦理和门徒训练的每个层面
- **[我推断]** 悖论的功能：
  - **认知冲击**：迫使听众停下自动化的思维
  - **记忆增强**：反常的表述比直白陈述更容易记住
  - **身份塑造**：跟随耶稣的人被定义为「与世界相反的人」
  - **现实重构**：不是在说「未来会翻转」，而是在宣告「真正的现实已经翻转了」

### 2.3 独特性分析

- **[学者分析]** 与同期拉比教导对比：犹太拉比也使用悖论（如希勒尔的「己所不欲，勿施于人」的否定式），但耶稣的悖论更极端、更系统，并且直接关联到 **他的个人身份**——不是因为律法如此说，而是因为「我告诉你们」
- **[我推断]** 其他古代哲人的悖论（如苏格拉底、第欧根尼的犬儒悖论）更多是智性挑战，而耶稣的悖论带有呼召性质——听了就要选择

## 3. 夸张法（Hyperbole）

### 3.1 主要夸张语句清单（[福音书文本呈现]）

| 夸张语句 | 出处 | 夸张域 |
|---------|------|--------|
| 「若是你的右眼叫你跌倒，就剜出来丢掉…若是右手叫你跌倒，就砍下来丢掉」 | 太5:29-30 | 身体自残 |
| 「骆驼穿过针的眼，比财主进神的国还容易呢」 | 太19:24；可10:25；路18:25 | 现实不可能 |
| 「你们若有信心像一粒芥菜种，就是对这座山说：『你从这边挪到那边』，它也必挪去」 | 太17:20；路17:6 | 自然不可能 |
| 「你们这假冒为善的文士和法利赛人有祸了！…你们把蠓虫滤出来，骆驼倒吞下去」 | 太23:23-24 | 比例夸张 |
| 「你们中间谁有百只羊失去一只，不把这九十九只撇在旷野去找那失去的羊？」 | 路15:4 | 数量反转 |
| 「你们祷告，无论求什么，只要信，就必得着」 | 太21:22 | 条件夸张 |
| 「凡看见妇女就动淫念的，这人心里已经与她犯奸淫了」 | 太5:28 | 内在化夸张 |
| 「你们要小心，不可将善事行在人的面前，故意叫他们看见」 | 太6:1 | 行为条件夸张 |

### 3.2 夸张法特征分析

- **[福音书文本呈现]** 耶稣的夸张有几个鲜明特点：
  1. **身体极端化**（挖眼、断手）：用极致意象强调道德严肃性
  2. **现实不可能**（骆驼穿针眼、移山）：用物理世界的荒谬强化属灵真理
  3. **比例失调**（滤出蠓虫吞下骆驼）：嘲弄宗教领袖的轻重倒置
  4. **内在化**（动淫念=犯奸淫）：将外部行为标准推向内心动机层面

- **[我推断]** 夸张法在耶稣的修辞工具箱中承担「紧迫性标记」功能——当他用夸张时，他在说：「这件事的重要性超越你正在用理性衡量的那个层次」

- **[学者分析]** 学术界主流认为这些不是字面命令（如早期教父如俄利根确曾主张字面解释挖眼），而是「修辞性夸张」（the Hyperbole View）。类比：古代希罗修辞手册中，夸张（hyperbole）被视为正当的修辞手段

- **[我推断]** 夸张的受众效果：
  - **震惊 → 暂停 → 反思**：极端表述让人不能滑过去
  - **争议性记忆**：两千年来「骆驼穿针眼」仍然是文化记忆，说明修辞持久性

### 3.3 夸张频率

- **[我推断]** 保守估计：正典福音书中至少有 15-20 处可明确归类为夸张的表述
- 分布上，马太福音的登山宝训段落最为密集

## 4. 高频词汇和句式模式

### 4.1 「我实在告诉你们」（ἀμὴν λέγω ὑμῖν）——签名式表达

- **[福音书文本呈现]** 统计：
  - 符类福音（马太、马可、路加）：**52 次**
  - 约翰福音：**25 次**（双重「实实在在」）
  - 合计：**77 次**

- **[学者分析]** 这是耶稣最具标志性的修辞手法。来自希伯来词根 א-מ-נ（确认、可靠、忠实）。在旧约传统中，「阿们」是**回应性**的——在别人说完后表示认同。但耶稣将其改为**先行性**的——在说话前就用它来宣告权威（Raymond Brown：这是「对他即将说出的话语来自父的确认」）

- **[学者分析]** 「在当代犹太实践中没有平行案例」——这是独特的耶稣修辞标记（Wikipedia, Amen）

- **[我推断]** 这个句式的功能：
  - **权威标记**：不需要引用拉比传统或经文来支持论点，自己的话语就是权威
  - **注意力捕获**：「听着，接下来这句话特别重要」
  - **身份宣告**：这个句式在很多使用场合直接或间接指向耶稣对自己的独特身份认知（如可14:25的「我不再喝这葡萄汁，直到我在神的国里喝新的那日子」——末世的自我定位）

### 4.2 「你们听见有话说……只是我告诉你们」（登山宝训的对仗句）

- **[福音书文本呈现]** 出现在太5:21-48，共六次对仗：
  1. 不可杀人 → 不可动怒（5:21-26）
  2. 不可奸淫 → 不可动淫念（5:27-30）
  3. 休妻需给休书 → 休妻除非淫乱（5:31-32）
  4. 不可背誓 → 什么誓都不可起（5:33-37）
  5. 以眼还眼 → 不抵抗恶人（5:38-42）
  6. 爱邻舍恨仇敌 → 爱仇敌（5:43-48）

- **[学者分析]** Robert Gundry 主张将其命名为「满全句」（Culminations）而非「反题」（Antitheses），因为耶稣不是反对律法，而是加深/完成/赋予律法新意

- **[福音书文本呈现]** 句式结构（希腊文本）：
  - `Ἠκούσατε ὅτι ἐρρέθη` (You have heard that it was said)...
  - `ἐγὼ δὲ λέγω ὑμῖν` (But I say to you)...

- **[我推断]** 这个句式的力量在于**引用与重写的差距**——耶稣站在传统之上，不是否认它，而是声称自己有权决定它的真正含义。这比直接否定更激进

### 4.3 「有耳可听的，就应当听」

- **[福音书文本呈现]** 出现位置：太11:15, 13:9, 13:43；可4:9, 4:23；路8:8, 14:35

- **[我推断]** 这个短句的功能类似**参与提示**（participation cue）：
  - 它提醒听众：你不是被动接收信息，你需要主动调动理解力
  - 在比喻语境中，它区分了「听见」和「听进去」
  - 在终末语境中（如启2-3的「圣灵向众教会所说的话，凡有耳的，就应当听」），它是紧迫性标记

### 4.4 用问题回答问题

- **[福音书文本呈现]** 典型例证：
  - 「你们说我是谁？」（太16:15）——先用问题试探门徒的理解水平，再宣告基督论
  - 「你的律法上写的是什么？你念的是怎样呢？」（路10:26）——反问律法师
  - 「这样，凯撒的物当归给凯撒，神的物当归给神」（太22:21）——用一个问题的答案回答另一个问题
  - 「约翰的洗礼是从天上来的，还是从人间来的？」（太21:25）——用反问题让敌人陷入两难
  - 「你们中间谁是没有罪的，谁就可以先拿石头打她」（约8:7）——用一个假设引出一个沉默

- **[我推断]** 耶稣的反问策略分析：
  - **防御型反问**：被试探时，用反问暴露对方动机（纳税给凯撒的问题）
  - **教学型反问**：引导学生自己得出答案（「你们说我是谁？」）
  - **审判型反问**：让人面对自己的内心（文士和法利赛人质问行淫妇人）
  - **苏格拉底式vs耶稣式**：区别在于苏格拉底要你理性自省，耶稣要你面对神

### 4.5 反问句

- **[福音书文本呈现]** 反问句极其频繁，尤其是「难道……吗？」「岂不……吗？」句式：
  - 「你们中间谁有儿子求饼，反给他石头呢？」（太7:9）
  - 「你们哪一个能用思虑使寿数多加一刻呢？」（太6:27）
  - 「一个仆人不能事奉两个主……你们不能又事奉神，又事奉玛门」（太6:24）
  - 「这星月之工岂不是要用石头打死吗？」（约10:33的引用）
  - 「你们这小信的人哪，风浪为什么这么大你们还怕呢？」（太8:26背后的反问语气）

- **[我推断]** 反问句的使用频率远超普通陈述句在一对一教学/辩论中。这使得耶稣的教导有一种**不容回避的紧迫感**——他很少给你「可能是A也可能是B」的开放性，而是用反问逼你看到唯一合理的答案

## 5. 引用模式

### 5.1 旧约引用概况

- **[福音书文本呈现]** 耶稣最常引用的旧约书卷：
  - **以赛亚书**（被他称为核心资源——路4:17-21引61:1-2；太13:14-15引6:9-10）
  - **诗篇**（被钉十字架时引用诗22:1；教导中引诗110:1）
  - **申命记**（旷野试探中三次引用申6-8章）
  - **何西阿书**（「我喜爱怜恤，不喜爱祭祀」，太9:13, 12:7引用何6:6）
  - **创世记**（论离婚时引用创1:27, 2:24）
  - **但以理书**（「人子」概念的直接来源，太24:15引但9:27, 11:31, 12:11）

- **[学者分析]** 学者发现耶稣的引用偏好与当时的拉比诠释方法不同：他引用旧约不是「以经解经」，而是「以自己解经」——经文的权威来自他对其的诠释

### 5.2 引用模式分类

| 模式 | 例子 | 出处 |
|------|------|------|
| 「经上记着说」（γέγραπται） | 旷野试探中三次 | 太4:4,7,10；路4:4,8,12 |
| 「你们没有念过吗？」 | 论安息日 | 太12:3-5；可2:25-26 |
| 「你们听见有话说……只是我告诉你们」 | 登山宝训 | 太5:21-48 |
| 「经上记着说……」+ 重新诠释 | 引用后给出新命令 | 可7:6-13 |
| 不直接引用，但明显指向旧约 | 「你们岂不知……？」暗示听众知道经文 | 多处 |

- **[我推断]** 引用模式的关键洞察：
  - **选择性引用**：耶稣几乎从不用旧约支持民族主义或战争文本（尽管约书亚记、士师记中有大量此类内容）
  - **聚焦伦理和怜悯**：引用集中在「神要什么——不是祭物，而是怜恤」
  - **引用作为框架，不是权威**：旧约是背景板，耶稣自己的话才是前景

### 5.3 「只是我告诉你们」——引用后的超越

- **[福音书文本呈现]** 这是最有影响力的引用策略：先引用（或承认传统），然后直接修改、超越、甚至反转
  - 太5:33-37：不可背誓 → 什么誓都不可起
  - 太5:38-42：以眼还眼 → 连左脸也转过来
  - 可7:9-13：以「各耳板」传统批评法利赛人

- **[我推断]** 这种模式非常大胆，因为它在形式上是**对比式引用**（引用A，给出非A），而「非A」来自耶稣本人的权威。这在当时的拉比犹太教中没有平行案例。

### 6.2 意象系统的一致性分析

- **[学者分析]** 耶稣的意象高度一致，跨多个教导语境复现，暗示这些意象有可能追溯到历史上的耶稣而非教会的编辑
- **[我推断]** 意象系统的结构逻辑：
  - **日常性**：所选意象都是第一世纪加利利农民/渔民/牧人的日常生活
  - **可扩展性**：简单意象容纳复杂含义（光不只是一个概念，挑战、审判、启示、使命都包含在内）
  - **互锁性**：意象之间相互连接（牧人和羊↔好牧人为羊舍命↔我是道路真理生命）

## 7. 确定性表达

### 7.1 「我告诉你们」——权威语气分析

- **[福音书文本呈现]** 耶稣使用「我告诉你们」的频率极高，且在使用方式上分为几种强度：

| 强度级别 | 句式 | 典型例子 |
|---------|------|---------|
| **最高** | 「我实在告诉你们」（Amen lego hymin） | 77处，见4.1节 |
| **高** | 「我告诉你们」（lego hymin） | 太5:20, 5:22, 5:28, 5:32, 5:34, 5:39, 5:44 等大量 |
| **中** | 「你们知道……但我要告诉你们」 | 登山宝训六次反题 |
| **隐含权威** | 直接以第一人称发布命令/呼召 | 「来跟从我」（太4:19）；「你的罪赦了」（可2:5） |

- **[学者分析]** 这是耶稣表达中最引人注目的特征：**他没有一次说「按照传统」或「拉比XX说」**。他的权威来自他自己。在同时代的犹太教导中，这是革命性的

### 7.2 什么时候他表达不确定？——客西马尼园的祷告

- **[福音书文本呈现]** 耶稣明确表达不确定/犹豫的时刻非常少，但有力：

  **客西马尼园（太26:39；可14:36；路22:42）：**
  > 「我父啊，倘若可行，求你叫这杯离开我。然而，不要照我的意思，只要照你的意思。」
  - 这是正典福音书中唯一一处耶稣对「会发生什么」表达愿望性不确定的地方
  - 注意修辞结构：**条件式请求**（倘若可行）+ **意愿顺服**（然而不要照我的意思）

  **十字架上（太27:46；可15:34）：**
  > 「我的神！我的神！为什么离弃我？」
  - 引用诗篇22:1。这是表达性不确定——不是教义上的不确定，而是经历被遗弃的体验性呼喊
  - 路加/约翰版本避免了这个表达（代之以「父啊，我将我的灵魂交在你手里」和「成了」）

  **关于末时（太24:36；可13:32）：**
  > 「但那日子，那时辰，没有人知道，连天上的使者也不知道，子也不知道，惟独父知道。」
  - 这是一个极为显眼的自我限制表达——耶稣明确说自己不知道某事
  - 被有些学者（如 Bart Ehrman）作为「低基督论」的原始传统证据

- **[我推断]** 这三个「不确定」时刻在修辞上形成了关键的**可信度锚点**：
  - 如果福音书作者是在编造耶稣话语，他们不会加入「子也不知道」这种损伤基督论的表达（即这极可能源于历史耶稣本人）
  - 耶稣的确定性不是全知型的（像某种超人全知），而是**使命型**的——他确定的是「我应当做什么」，而不是「我知道一切」
  - 客西马尼园的表达是**二段式确定**：对事件不确定（这杯能不能离开我），但对使命确定（照你的意思）

### 8.2 情感表达特征分析

- **[福音书文本呈现]** 耶稣的情感表达**从不失控**——即使是愤怒（洁净圣殿）也是有目的、有焦点的行动，而非情绪爆发
- **[福音书文本呈现]** 情感总是与**教导目的**绑定——情感不是为表达而表达，而是作为教学工具

  - 哭泣 → 启示对世人的爱和即将到来的审判（为耶路撒冷哭）
  - 怜悯 → 示范恩慈怎样超越律法（安息日治病）
  - 喜乐 → 在成功服事后以感恩转向父（路10:21）

- **[我推断]** 情感表达的节制与「父的旨意」意识密切相关——耶稣的情感表达总是被他对自己使命的认知所约束。用现代心理学术语来说，他的情感调节能力极强

## 9. 综合分析与模式总结

### 9.1 耶稣表达风格的三层结构

```
表层：      比喻 + 夸张 + 悖论    ← 被大众听到
中层：      反问 + 意象系统       ← 需要思考才能进入
深层：      「我实在告诉你们」    ← 权威宣示 + 身份宣告
```

- **[福音书文本呈现]** 听众可以停留在任何一层——有人只听到故事（表层），有人开始思考（中层），有人接受了耶稣的权柄并跟随（深层）
- **[我推断]** 这是一个**分级卷入系统**（layered engagement system）：用一种表达方式服务多种受众层次

### 9.2 核心修辞模式矩阵

| 特征 | 频率 | 独特性 | 持久影响力 | 受众接受难度 |
|------|------|--------|-----------|------------|
| 比喻 | 很高 | 高（但有无传统基础） | 极高 | 低（表面理解）→ 高（深度理解） |
| 悖论 | 高 | 极高（系统性应用） | 极高 | 中（需认知翻转） |
| 夸张 | 中高 | 中（先知也有）但风格极端 | 高 | 低（震撼但易记） |
| 「我实在告诉你们」 | 很高(77次) | 极高（无犹太平行） | 极高 | 低→高（取决于接受权威程度） |
| 反问 | 很高 | 中（也是拉比常用） | 中高 | 低（迫使自省） |
| 引用旧约+超越 | 中 | 极高（自我权威性） | 中高 | 中（需上下文） |

### 9.3 对比：与其他古代教师/哲人的区别

| 维度 | 耶稣 | 苏格拉底 | 孔子 | 佛佗 | 拉比传统 |
|------|------|---------|------|------|---------|
| 权威来源 | 自己 | 理性/神谕 | 传统/礼 | 觉悟 | 律法/传统 |
| 核心教法 | 比喻+悖论 | 辩证法 | 格言+典故 | 譬喻+论辩 | 解经+判例 |
| 听众关系 | 呼召(跟随) | 对话(辩论) | 教导(学习) | 修行(离苦) | 教导(遵守) |
| 情绪表达 | 丰富但节制 | 理性主导 | 含蓄 | 平静 | 各拉比不同 |
| 确定性 | 极高 | 反讽式谦虚 | 谦逊 | 确定但开放 | 依赖于经文 |

### 9.4 「耶稣表达DNA」的终极特征

**[我推断]** 从以上所有维度综合提炼：

1. **以故事启动思考，非以命题启动** —— 他不是先说「神的国是这样的」，而是先讲一个故事让人自己得出结论
2. **以翻转冲击认知，非以渐进说服** —— 悖论不是为了让你理解，而是为了让你停下来
3. **以短句锚定记忆，非以论证巩固** —— 「骆驼穿过针的眼」比一篇论文更有生命力
4. **以反问取代告知，非以答案终结** —— 「你们说我是谁？」「律法上写的是什么？」「你读的是怎样呢？」
5. **以自我为权威锚点，非以传统为参照** —— 「我告诉你们」不仅是一个句式，是整个表达系统的哲学基础
6. **以情感激活，非以冷静中立** —— 他的愤怒、怜悯、哭泣、欢乐都是教导的一部分
7. **以间隙邀请参与，非以满负荷灌输** —— 比喻不解释就意味留下了空间让听众参与意义的构建：「有耳可听的，就应当听」
